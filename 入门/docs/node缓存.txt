--简介
    --什么是buffer
        指的是内存区域
        表示v8引擎外部分配的固定大小的内存块
        可视为整数数组,每个整数代表一个数据字节
        由Buffer模块提供技术支持
    --为什么需要buffer
        被引入用以帮助开发者处理二进制数据
        与流紧密相连。 当流处理器接收数据的速度快于其消化的速度时，则会将数据放入 buffer 中
--使用
    --创建buffer
        ` Buffer.from()     http://nodejs.cn/api/buffer.html#buffer_buffer_from_buffer_alloc_and_buffer_allocunsafe
            Buffer.from(array)
            Buffer.from(arrayBuffer[, byteOffset[, length]])
            Buffer.from(buffer)
            Buffer.from(string[, encoding])
        ` Buffer.alloc()    http://nodejs.cn/api/buffer.html#buffer_class_method_buffer_alloc_size_fill_encoding
            参数为数值,表示字节大小
            会初始格式化化
        ` Buffer.allocUnsafe()
            参数为数值,表示字节大小
            不会初始格式化化
    --使用buffer
        ` 访问buffer的内容
            像数组一样被访问
            使用 toString() 方法打印 buffer 的全部内容
        ` 获取buffer的长度
            使用 length 属性
        ` 迭代buffer的内容
            使用循环
        ` 更改 buffer 的内容
            使用 write() 方法将整个数据字符串写入 buffer
            使用数组下标更改的方法
        ` 复制 buffer
            使用copy()方法复制 buffer
                [buffer标识,定义开始位置,结束位置,新的 buffer 长度]
        ` 切片 buffer
            使用 slice() 方法创建它。 
                [起始位置,结束位置]
